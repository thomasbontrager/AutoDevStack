# Plugins

This directory is reserved for the AutoDevStack plugin system.

Plugins allow community members and teams to extend AutoDevStack with custom templates, generators, and post-scaffold hooks.

## Plugin System (Coming in v2.0)

The plugin system is planned for v2.0. See [ROADMAP.md](../ROADMAP.md) for details.

## Contributing a Plugin

Once the plugin API is stable, you will be able to:

1. Create a plugin package following the plugin spec
2. Register it locally or publish it to npm
3. Use it via `autodevstack --plugin <name>`
