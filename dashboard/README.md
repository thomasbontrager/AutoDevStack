# AutoDevStack Dashboard

A simple developer dashboard MVP for viewing and managing projects generated by the AutoDevStack CLI.

## Features

- **Login page** (`/login`) — mock authentication (see demo credentials below)
- **Dashboard** (`/dashboard`) — overview of project stats, recent projects, and stacks used
- **Projects** (`/projects`) — full project list with add, search, and delete

## Getting Started

```bash
cd dashboard
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Project Storage

Projects are stored in the browser's `localStorage` under the key `autodevstack_projects`. The dashboard is seeded with three sample projects on first load.
